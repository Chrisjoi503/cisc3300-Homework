1. Classes are __templates____ for creating objects. 
2. The ____visbility ___ of a property, a method or a constant can be defined by prefixing the declaration with the keywords public, protected or private. 
3. When a class derives from another class is called ___inheritence___ 
4. __Namespaces__ allow the PHP interpreter to tell the difference between two classes, functions, or constants that share the same name. 
5. Several predefined variables in PHP are _superglobals_, which means they are available in all scopes throughout a script. 
6. _____Builtins___ can be called directly, from within a script, to perform a specific task. 
7. The ___header___ function sends a raw HTTP header to a client. 
8. ____MVC__ is an architectural/design pattern that separates an application into three main logical components Model, View, and Controller. 
9. __Libraries__ allow developers to use code that they, or other programmers, have already written to perform a task.
